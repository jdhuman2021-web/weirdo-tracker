## Multi-Agent System Architecture - BUILT

**Status:** Core architecture complete with dynamic token support

### Components Built

**1. RESEARCH Agent**
- Reads token list from `config/tokens.json`
- Fetches live data from DexScreener API
- Rate limiting: 6 seconds between requests
- Output: `data/latest.json`

**2. THINKING Agent**
- Scores opportunities 0-100
- Algorithm:
  - Price drop >30%: 30 pts
  - Volume spike 3x: 25 pts
  - Liquidity >$100K: 15 pts
  - Micro-cap < $1M: 20 pts
- Output: `data/opportunities.json`

**3. ALERT Agent**
- Checks thresholds (score >80, price drop >30%)
- Saves alerts to `data/alerts.json`
- Ready for Telegram integration

**4. Token Manager**
- Interactive PowerShell tool
- Add/remove tokens from config
- Tracks: symbol, name, address, chain, added_date, source, notes
