## Telegram Diagnosis (22:17 UTC)

### Problem
Telegram channel not responding. User reported issue at 22:14 UTC.

### Root Cause Found
```
gateway.bind = loopback
```
Gateway only bound to 127.0.0.1 — external channels (Telegram) cannot reach it.

### Fix Applied
User ran: `openclaw config set gateway.bind=lan` + `openclaw gateway restart`

### Status
- Webchat working (this session)
- Telegram should now work after restart
- Gateway: Ready state
